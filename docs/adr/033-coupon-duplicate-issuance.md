# ADR-033: 동일 유저에게 동일 템플릿 쿠폰 중복 발급 허용

- 날짜: 2026-06-09
- 상태: 승인됨 (재검토 예정)

## 결정

같은 유저에게 동일 쿠폰 템플릿의 쿠폰을 여러 장 발급하는 것을 **허용**한다.
`(user_id, coupon_template_id)` 복합 UNIQUE 제약을 적용하지 않는다.

## 근거

현재 요구사항에 중복 발급 방지 정책이 명시되어 있지 않다.
제약을 추가하는 것은 언제든 가능하나, 허용 방향에서 불허 방향으로의 전환은 데이터 정합성 검토와 마이그레이션이 필요하다.
현 시점에서는 어드민의 책임하에 발급하는 것으로 간주하고 제약을 두지 않는다.

## 향후 고려사항

- 중복 발급 방지 정책이 요구사항에 추가될 경우, `UNIQUE(user_id, coupon_template_id)` DB 제약 및 서비스 레이어 검증을 추가한다.
- 이 결정은 요구사항 변경 시 재검토한다.

# ADR-004: 상품 응답에 브랜드명 포함

- 날짜: 2026-05-20
- 상태: 승인됨

## 결정

상품 조회 응답(`GET /api/v1/products`, `GET /api/v1/products/{productId}`)에 `brandId`와 함께 `brandName`을 포함한다.

## 근거

**요구사항 관점**: 요구사항 문서에 "상품, 브랜드 정보 중 고객과 어드민에게 제공되어야 할 정보에 대해 고민해보세요"라고 명시되어 있다. 상품 등록 시 브랜드를 반드시 지정해야 하므로, 브랜드는 상품의 핵심 속성이다. `brandId`만 반환할 경우 클라이언트가 브랜드명을 표시하기 위해 추가 API 호출이 필요하며 이는 N+1 문제로 이어진다.

**REST 설계 관점**: REST 원칙상 리소스는 자기 서술적(self-descriptive)이어야 한다. ID만 반환하면 클라이언트가 의미를 파악하기 어렵다.

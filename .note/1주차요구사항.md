# 1주차 요구사항

## 기능 구현

- 회원가입
- 내 정보 조회
- 비밀번호 수정

## 공통 요청 헤더

회원가입 이후, 유저 정보가 필요한 모든 요청은 아래 헤더를 통해 요청한다.

- `X-Loopers-LoginId`: 로그인 ID
- `X-Loopers-LoginPw`: 비밀번호

## 회원가입

### 필요 정보

- 로그인 ID
- 비밀번호
- 이름
- 생년월일
- 이메일

### 요구사항

- 이미 가입된 로그인 ID로는 가입할 수 없다.
- 각 정보는 포맷에 맞게 검증해야 한다.
- 로그인 ID는 영문과 숫자만 허용한다.
- 비밀번호는 암호화해 저장한다.

### 비밀번호 규칙

- 8~16자의 영문 대소문자, 숫자, 특수문자만 허용한다.
- 생년월일은 비밀번호에 포함될 수 없다.

## 내 정보 조회

### 반환 정보

- 로그인 ID
- 이름
- 생년월일
- 이메일

### 요구사항

- 이름은 마지막 글자를 마스킹해서 반환한다.
- 마스킹 문자는 `*`로 통일한다.

## 비밀번호 수정

### 필요 정보

- 기존 비밀번호
- 새 비밀번호

### 요구사항

- 새 비밀번호는 비밀번호 규칙을 따라야 한다.
- 새 비밀번호는 현재 비밀번호와 같을 수 없다.

### 비밀번호 규칙

- 8~16자의 영문 대소문자, 숫자, 특수문자만 허용한다.
- 생년월일은 비밀번호에 포함될 수 없다.
